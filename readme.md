## Deskripsi Project
Project ini merupakan form input sederhana menggunakan PHP.
Data yang diinput:
- Firstname
- Lastname
- Phone Number
- Address

Setelah tombol submit ditekan, data akan ditampilkan di halaman.

## File Project
- `index.php` : halaman utama
- `class.php` : class Biodata
- `style.css` : tampilan form
- `README.md` : dokumentasi project

## Cara Menjalankan
1. Simpan folder project di `htdocs`
2. Jalankan Apache di XAMPP
3. Buka browser
4. Akses `http://localhost/Project_BasedTask_Jenri`

Project ini menggunakan:
- HTML
